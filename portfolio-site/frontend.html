security

Route Guards

Functional guards protect routes based on authentication status and user roles. Redirects to login if unauthorized.

export const authGuard: CanActivateFn =
  (route, state) => {
    const auth = inject(AuthService);
    if (auth.isAuthenticated()) {
      return true;
    }
    return inject(Router)
      .createUrlTree(['/login']);
  };
http

HTTP Interceptors

Functional interceptors add JWT tokens to requests and handle 401 errors globally with automatic token refresh.

export const authInterceptor:
  HttpInterceptorFn = (req, next) => {
    const token = inject(TokenService)
      .getToken();
    if (token) {
      req = req.clone({
        setHeaders: {
          Authorization: `Bearer ${token}`
        }
      });
    }
    return next(req);
  };

State Management with Signals

Auth Service Example

@Injectable({ providedIn: 'root' })
export class AuthService {
  // Private writable signal
  private readonly _currentUser =
    signal<User | null>(this.loadFromStorage());

  // Public readonly signals
  readonly currentUser = this._currentUser.asReadonly();

  readonly isAuthenticated = computed(() =>
    !!this._currentUser() &&
    !this.tokenService.isTokenExpired()
  );

  readonly userRoles = computed(() =>
    this._currentUser()?.roles ?? []
  );

  readonly isAdmin = computed(() =>
    this.hasRole('Administrator')
  );

  // Update state
  login(credentials: LoginRequest) {
    return this.http.post<AuthResponse>(url, credentials)
      .pipe(
        tap(response => {
          this._currentUser.set(response.user);
          this.saveToStorage(response.user);
        })
      );
  }

  logout() {
    this._currentUser.set(null);
    this.clearStorage();
  }
}

Why Signals?

  • check_circle
    Simpler Than RxJS

    No subscriptions to manage, no memory leaks from forgotten unsubscribes

  • check_circle
    Fine-Grained Reactivity

    Only components using a signal re-render when it changes

  • check_circle
    Computed Values

    Derived state automatically updates when dependencies change

  • check_circle
    Type Safe

    Full TypeScript support with inferred types

  • check_circle
    Synchronous

    Read values synchronously with signal() - no async/await needed
